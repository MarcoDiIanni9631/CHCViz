# CHCViz

This repository contains the code to transform a Smart Contract (or a CHC Prolog file) into a Predicate dependency graph.

## Features

The toolchain supports two modes:

### From Solidity (.sol)
Full pipeline:
1. Compiles a Solidity contract into an SMT-LIB query using SolCMC.
2. Translates the SMT-LIB query into Prolog Eldarica.
3. Analyzes the resulting Prolog code, resolving any issues related to variables and predicates.
4. Converts the analyzed Prolog code into a .dot file using Logtalk.
5. Generates an SVG diagram from the .dot file using Graphviz.

### From Prolog (.pl)
Partial pipeline (steps 4–5 only):
1. Strips `:-` directives (pred, mode, query) from the Prolog file.
2. Converts the cleaned Prolog code into a .dot file using Logtalk.
3. Generates an SVG diagram from the .dot file using Graphviz.

This mode is useful when you already have a CHC Prolog file (e.g., a `.t.pl` generated by another tool) and want to visualize the predicate dependency graph directly.

## Dependencies

Before running the toolchain, make sure you have the following software installed on your system:

- SOLC
- Eldarica
- Logtalk
- Graphviz
- SwiProlog

## Usage

Navigate to the directory containing the repository, then:

**From a Solidity file:**
```
python3 chcviz.py <solidity_file.sol>
```

**From a Prolog CHC file:**
```
python3 chcviz.py <prolog_file.pl>
```

**From a folder (all .sol files):**
```
python3 chcviz.py -d <directory_path>
python3 chcviz.py -r <directory_path>   # recursive
```

The SVG output is saved in a `dot_dias/` subfolder in the same directory as the input file.
